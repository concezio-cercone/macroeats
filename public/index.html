How the macros work — read this once
Some restaurants have ✓ real macros (verified from the chain's official nutrition). Most discovered places show * estimated macros — educated guesses from the cuisine type, not from that specific restaurant. Estimates can be off, so treat them as ballpark. When you know the real numbers, hit ✏️ Fix macros on any meal — it saves them permanently and every future search uses them. The tool gets more accurate the more you use it.
